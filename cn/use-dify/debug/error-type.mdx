每个节点类型都会抛出特定的错误类别，帮助你了解出错的原因以及如何修复。

## 节点特定错误

<Tabs>
<Tab title="Code" icon="code">

`CodeNodeError`
<Info>你的 Python 或 JavaScript 代码在执行过程中抛出了异常</Info>

![Code Error](https://assets-docs.dify.ai/2024/12/c86b11af7f92368180ea1bac38d77083.png)

`OutputValidationError`
<Info>你的代码返回的数据类型与你配置的输出变量类型不匹配</Info>

`DepthLimitError`
<Info>你的代码创建的嵌套数据结构深度超过了 5 层</Info>

`CodeExecutionError`
<Info>沙箱服务无法执行你的代码 - 通常意味着服务已停止</Info>

![](https://assets-docs.dify.ai/2024/12/ab8cae01a590b037017dfe9ea4dbbb8b.png)

</Tab>
<Tab title="LLM" icon="brain">

`VariableNotFoundError`
<Info>你的提示词模板引用了一个在工作流上下文中不存在的变量</Info>

![](https://assets-docs.dify.ai/2024/12/f20c5fbde345144de6183374ab277662.png)

`InvalidContextStructureError`
<Info>你向上下文字段传递了数组或对象，该字段仅接受字符串</Info>

`NoPromptFoundError`
<Info>提示词字段完全为空</Info>

`ModelNotExistError`
<Info>在大型语言模型节点配置中未选择模型</Info>

`LLMModeRequiredError`
<Info>所选模型未配置有效的 API 凭据</Info>

`InvalidVariableTypeError`
<Info>你的提示词模板不是有效的 Jinja2 语法或纯文本格式</Info>

![](https://assets-docs.dify.ai/2024/12/9882f7a5ee544508ba11b51fb469a911.png)

</Tab>
<Tab title="HTTP Request" icon="globe">

`AuthorizationConfigError`
<Info>API 端点缺失或身份验证配置无效</Info>

`InvalidHttpMethodError`
<Info>HTTP 方法必须是 GET、HEAD、POST、PUT、PATCH 或 DELETE</Info>

`ResponseSizeError`
<Info>API 响应超过了 10MB 大小限制</Info>

`FileFetchError`
<Info>无法检索请求中引用的文件变量</Info>

`InvalidURLError`
<Info>URL 格式错误或无法访问</Info>

</Tab>

`ToolFileError`

`ToolInvokeError`

## 系统级错误

`InvokeConnectionError`
<Info>与外部服务的网络连接失败</Info>

`InvokeServerUnavailableError` 
<Info>外部服务返回 503 状态或暂时停机</Info>

`InvokeRateLimitError`
<Info>你已达到 API 或模型提供商的速率限制</Info>

`QuotaExceededError`
<Info>你对此服务的使用配额已超限</Info>
