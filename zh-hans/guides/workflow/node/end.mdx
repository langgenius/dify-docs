---
title: 输出
---

<Info>
    - 输出节点之前称为「结束」节点。与结束节点不同的是，输出节点在 Workflow 中是 *可选* 的，仅用于向终端用户显式输出数据。

    - 此节点仅适用于 Workflow 应用，而 Chatflow 则使用 [直接回复](/zh-hans/guides/workflow/node/answer) 节点在对话流中返回信息。
</Info>

在输出节点中，你可以通过添加输出变量来定义向终端用户返回的内容，例如 LLM 生成的响应。需指定至少一个输出变量；否则，不会返回任何内容。

当作为后端服务 API 对外公开时，没有输出节点的 Workflow 将不会向 API 调用方返回任何值。
