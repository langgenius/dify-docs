---
title: New Agent API
sidebarTitle: Overview
description: Chat-style API for New Agent apps, where the model reasons and uses tools autonomously
---

New Agent apps follow the chat message flow: send a message, stream the reply, and manage conversations through the same endpoints as other chat-style apps. Responses are streaming-only, and the stream carries plain `message` events; the model's reasoning and tool activity are not exposed over the API.

<Note>
The New Agent app type is in beta. File inputs on [Send Chat Message](/en/api-reference/chats/send-chat-message) are not yet supported for New Agent apps.
</Note>
