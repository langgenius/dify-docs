New Agent アプリはチャットメッセージのフローに従います。メッセージを送信し、回答をストリーミングで受け取り、他のチャット形式アプリと同じエンドポイントで会話を管理します。レスポンスはストリーミングのみで、ストリームには通常の `message` イベントだけが含まれます。モデルの推論やツール使用は API では公開されません。

<Note>
New Agent アプリタイプは現在ベータ版です。[チャットメッセージを送信](/ja/api-reference/chat-messages/send-chat-message) では、New Agent アプリへのファイル入力は未対応です。
</Note>
