キーボードショートカットでワークフロー構築を高速化します。

<Tip>
**[Go to Anything](/ja/self-host/use-dify/build/goto-anything)**：Dify 内の任意の場所で `Cmd+K`（macOS）または `Ctrl+K`（Windows）を押すと、ほぼあらゆるものを検索してジャンプできます。

- `@` を入力して、アプリ、インテグレーション、ナレッジベース、ワークフローノードを検索します。
- `/` を入力して、`/docs`、`/theme`、`/language` などのコマンドを実行します。

## 選択と編集

キャンバス上で 1 つ以上のノードを選択している状態で：

| Windows | macOS | 操作 |
|:--------|:------|:-----|
| `Ctrl` + `C` | `Cmd` + `C` | コピー |
| `Ctrl` + `V` | `Cmd` + `V` | 貼り付け |
| `Ctrl` + `D` | `Cmd` + `D` | 複製 |
| `Delete` | `Delete` | 選択したノードまたは接続線を削除 |
| `Shift`（長押し） | `Shift`（長押し） | 選択したノードの変数のつながりをハイライト |

## キャンバスモード

| Windows | macOS | 操作 |
|:--------|:------|:-----|
| `V` | `V` | ポインターモード（選択） |
| `H` | `H` | ハンドモード（パン） |
| `C` | `C` | コメントモード |

## ズームとレイアウト

| Windows | macOS | 操作 |
|:--------|:------|:-----|
| `Ctrl` + `1` | `Cmd` + `1` | ビューに合わせる |
| `Ctrl` + `=` | `Cmd` + `=` | ズームイン |
| `Ctrl` + `-` | `Cmd` + `-` | ズームアウト |
| `Shift` + `1` | `Shift` + `1` | 100% にズーム |
| `Shift` + `5` | `Shift` + `5` | 50% にズーム |
| `Ctrl` + `O` | `Cmd` + `O` | ノードを自動整列 |
