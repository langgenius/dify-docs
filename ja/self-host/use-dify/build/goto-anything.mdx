**Go to Anything（グローバル検索）** は、コンソール全体を対象にした検索ボックスです。一度入力するだけで、アプリ、ナレッジベース、マーケットプレイスのインテグレーション、編集中のキャンバス上のノードを見つけられます。クイックコマンドも実行できます。

### オーバーレイを開く

macOS では `⌘ + K`、Windows/Linux では `Ctrl + K` を押すか、サイドバー上部の検索ボタンをクリックします。

何も入力していない状態では、最近開いたアプリとナレッジベースが表示されるため、すぐに戻れます。

<Info>
ショートカットが反応しない場合、ほかのアプリやブラウザ拡張機能がショートカットを使用している可能性があります。サイドバーのボタンは常に使えます。
</Info>

### @ で検索

キーワードを入力するとすべてを横断検索し、結果はカテゴリごとに表示されます。スコープを付けると検索対象を絞り込めます。

- `@app` – アプリを検索
- `@knowledge` または `@kb` – ナレッジベースを検索
- `@plugin` – マーケットプレイスのインテグレーションを検索
- `@node` – 開いているキャンバス内のノードを検索（ワークフローとナレッジパイプラインで利用可能）

矢印キーで結果を移動し、`Enter` で開きます。

### / でコマンドを実行

`/` を入力するとすべてのコマンドが表示され、続けて入力すると絞り込めます。

| コマンド | 動作 |
|:---|:---|
| `/create` / `/new` | アプリの説明から Workflow または Chatflow を生成。種類は自分で指定するか、**自動** で AI に任せる |
| `/refine` / `/improve` | 説明した変更を、編集中の Workflow または Chatflow に適用（開いているときのみ） |
| `/go` | コンソールの主要セクションへ移動 |
| `/docs` | ヘルプドキュメントを開く |
| `/community` | Discord コミュニティを開く |
| `/account` | アカウントページへ移動 |
| `/theme` | インターフェースのテーマを切り替える |
| `/language` / `/lang` | インターフェースの言語を切り替える |
