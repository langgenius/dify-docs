  <div style={{flex: 3, minWidth: '300px'}}>
    公開が完了すると、以下の操作が可能です：<br/><br/>

    **ドキュメントの追加（Go to add documents）**<br/>
    このオプションをクリックすると、ナレッジベースのデータソース選択画面に移動し、直接ドキュメントをアップロードできます。<br/><br/>

    **API へのアクセス（Access API Reference）**<br/>
    API ドキュメントページに移動し、ナレッジベースの API 呼び出し方法や利用手順を確認できます。<br/><br/>

    **ナレッジパイプラインとして公開（Publish as a Knowledge Pipeline）**<br/>
    任意で **ナレッジパイプラインとして公開** を使用すると、このパイプラインを再利用可能なテンプレートとして保存できます。テンプレートはカスタマイズ済みセクションに表示され、将来再利用できます。
