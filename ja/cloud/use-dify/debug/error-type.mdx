## システムレベルのエラー

`InvokeConnectionError`
<Info>外部サービスへのネットワーク接続が失敗しました</Info>

`InvokeServerUnavailableError` 
<Info>外部サービスが 503 ステータスを返したか、一時的に停止しています</Info>

`InvokeRateLimitError`
<Info>API またはモデルプロバイダーのレート制限に達しました</Info>

`QuotaExceededError`
<Info>このサービスの使用量クォータを超過しました</Info>
