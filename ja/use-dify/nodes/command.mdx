Agent ノードはコマンドラインを自律的に実行できますが、実行するコマンドが明確で LLM の推論が不要な場合は、コマンド実行ノードの方が適しています。より高速で、トークンを消費しません。

コマンドはデフォルトでルートディレクトリで実行されます。別の作業ディレクトリを使用するには、既存のパスまたは実行中に上流ノードによって作成されるパスを入力してください。
