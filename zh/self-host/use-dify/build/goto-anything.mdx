**全局搜索（Go to Anything）** 是覆盖整个控制台的搜索框：输入一次即可找到应用、知识库、市场中的集成，以及当前编辑画布上的节点，还可以运行快捷命令。

### 打开方式

在 macOS 上按 `⌘ + K`，在 Windows/Linux 上按 `Ctrl + K`，或点击侧边栏顶部的搜索按钮。

尚未输入时，面板会先列出你最近打开过的应用和知识库，一次按键即可返回。

<Info>
如果快捷键没有反应，可能是其他应用或浏览器扩展占用了它。侧边栏按钮始终可用。
</Info>

### 使用 @ 搜索

输入关键词即可搜索全部内容，结果按类别分组。加上范围前缀可缩小搜索：

- `@app`：搜索应用
- `@knowledge` 或 `@kb`：搜索知识库
- `@plugin`：搜索市场中的集成
- `@node`：在当前打开的画布中搜索节点（工作流和知识流水线中可用）

使用上下箭头选择结果，按 `Enter` 打开。

### 使用 / 运行命令

输入 `/` 可查看全部命令，继续输入即可筛选：

| 命令 | 作用 |
|:---|:---|
| `/create` / `/new` | 描述你想要的应用，Dify 会将其生成为 Workflow 或 Chatflow；可自行指定类型，或选择 **自动**，让 AI 决定 |
| `/refine` / `/improve` | 描述一处修改，Dify 会将其应用到你正在编辑的 Workflow 或 Chatflow（仅在打开时可用） |
| `/go` | 跳转到控制台的主要板块 |
| `/docs` | 打开帮助文档 |
| `/community` | 打开 Discord 社区 |
| `/account` | 前往账户页面 |
| `/theme` | 切换界面主题 |
| `/language` / `/lang` | 切换界面语言 |
