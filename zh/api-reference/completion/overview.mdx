文本生成应用运行在 `completion` 模式下，每次调用各自提交输入值并返回一条回复，调用之间不保留会话状态。这类应用适合翻译、文章撰写、摘要总结等一次性任务，每次请求都相互独立。

## 生成文本

[发送消息](/zh/api-reference/completion-messages/send-completion-message) 将 `inputs` 中的取值填入应用的提示词模板并生成回复。将 `response_mode` 设为 `streaming`，可持续接收 `message` 事件，直至收到 `message_end`（开启自动播放时还会收到 `tts_message` 事件）；设为 `blocking`，则需等待完整结果返回。

[停止响应](/zh/api-reference/completion-messages/stop-completion-message-generation) 通过 `task_id` 取消运行中的任务，但仅适用于 `streaming` 模式；`blocking` 模式下调用返回时任务已执行完毕，无法取消。

## 处理文件

[上传文件](/zh/api-reference/files/upload-file) 用于上传图片、文档、音频或视频文件，供 [发送消息](/zh/api-reference/completion-messages/send-completion-message) 引用，且仅限上传者本人使用。[下载文件](/zh/api-reference/files/download-file) 可预览或下载通过该方式上传的文件，前提是该文件属于当前应用内的某条消息。

当某个响应只返回终端用户 ID（例如上传文件响应中的 `created_by`）而非完整信息时，可通过 [获取终端用户信息](/zh/api-reference/end-users/get-end-user-info) 查询详情。

## 语音转写与合成

[语音转文字](/zh/api-reference/audio/convert-audio-to-text) 可将上传的音频文件（MP3、M4A、WAV、AMR 或 MPGA，最大 30 MB）转写为文字，让终端用户用语音代替打字输入。[文字转语音](/zh/api-reference/audio/convert-text-to-audio) 可将生成的回复合成为语音。

## 获取应用信息与配置

[获取应用基本信息](/zh/api-reference/applications/get-app-info) 返回应用的名称、描述、标签和模式。[获取应用参数](/zh/api-reference/applications/get-app-parameters) 返回输入表单配置：功能开关、输入参数名称、类型及默认值，可用于构建客户端界面。

[获取应用元数据](/zh/api-reference/applications/get-app-meta) 返回工具图标等配置元数据。[获取应用 WebApp 设置](/zh/api-reference/applications/get-app-webapp-settings) 返回 WebApp 的站点配置、主题和自定义选项。

## 收集反馈

终端用户可通过 [提交消息反馈](/zh/api-reference/feedback/submit-message-feedback) 将回复评为 `like` 或 `dislike`，并附加文字说明；将 `rating` 传入 `null` 可撤销此前的评价。[获取应用的消息反馈](/zh/api-reference/feedback/list-app-feedbacks) 返回应用内终端用户和管理员提交的全部反馈。
