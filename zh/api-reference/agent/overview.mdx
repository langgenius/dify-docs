New Agent 应用遵循对话消息流程：发送消息、流式接收回复，并使用与其他对话型应用相同的接口管理会话。响应仅支持流式返回，流中只包含普通的 `message` 事件；模型的推理和工具调用过程不会通过 API 暴露。

<Note>
New Agent 应用类型目前处于 Beta 阶段。[发送对话消息](/zh/api-reference/chats/send-chat-message) 暂不支持为 New Agent 应用传入文件。
</Note>
