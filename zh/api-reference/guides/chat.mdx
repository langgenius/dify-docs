[聊天助手应用](/zh/cloud/use-dify/build/chatbot)（`chat` 模式）和 Agent 应用（`agent-chat` 模式）共用这套 API：同一组接口用于发送消息、管理会话、处理文件和获取应用设置。两者仅在回复的流式事件上有所不同：聊天助手应用以 `message` 事件流式返回回复，Agent 应用则返回 `agent_thought` 和 `agent_message` 事件。

## 发送消息并接收流式回复

- **[发送对话消息](/zh/api-reference/chat-messages/send-chat-message)**：向聊天助手或 Agent 应用发送用户提问。阻塞式返回在回复完成后一次性返回全部内容；Agent 应用仅支持流式返回。
- **[停止响应](/zh/api-reference/chat-messages/stop-chat-message-generation)**：在回复完成前中断流式输出。
- **[获取下一轮建议问题列表](/zh/api-reference/chat-messages/get-next-suggested-questions)**：回复完成后，基于当前对话内容推荐后续问题。

## 管理会话

通过 API 创建的会话，与该应用 WebApp 中发起的会话相互隔离。

- **[获取会话列表](/zh/api-reference/conversations/list-conversations)**：按最近活跃时间排序，返回当前用户的会话。
- **[获取会话历史消息](/zh/api-reference/conversations/list-conversation-messages)**：返回某个会话的历史消息，适合滚动加载的聊天界面。
- **[重命名会话](/zh/api-reference/conversations/rename-conversation)**：设置或自动生成会话名称。
- **[删除会话](/zh/api-reference/conversations/delete-conversation)**：删除指定会话。
- **[获取对话变量](/zh/api-reference/conversations/list-conversation-variables)**：读取会话在多轮对话间保留的变量值。
- **[更新对话变量](/zh/api-reference/conversations/update-conversation-variable)**：直接修改其中某个变量的值。

## 处理文件

- **[上传文件](/zh/api-reference/files/upload-file)**：上传图片、文档、音频或视频文件，供 [发送对话消息](/zh/api-reference/chat-messages/send-chat-message) 引用。文件仅限上传的终端用户使用。
- **[下载文件](/zh/api-reference/files/download-file)**：预览或下载已上传的文件，前提是该文件属于当前应用内的某条消息。
- **[获取终端用户信息](/zh/api-reference/end-users/get-end-user-info)**：根据终端用户 ID（例如上传文件响应中的 `created_by`）查询该用户的详细信息。

## 语音转写与合成

- **[语音转文字](/zh/api-reference/audio/convert-audio-to-text)**：将上传的音频文件（MP3、M4A、WAV、AMR 或 MPGA，最大 30 MB）转写为文字，让终端用户用语音代替打字输入。
- **[文字转语音](/zh/api-reference/audio/convert-text-to-audio)**：将回复合成为语音。

## 获取应用信息与配置

- **[获取应用基本信息](/zh/api-reference/applications/get-app-info)**：返回应用的名称、描述、标签和模式。
- **[获取应用参数](/zh/api-reference/applications/get-app-parameters)**：返回输入表单配置（功能开关、参数名称、类型及默认值），可用于构建客户端界面。
- **[获取应用元数据](/zh/api-reference/applications/get-app-meta)**：返回工具图标等配置元数据。
- **[获取应用 WebApp 设置](/zh/api-reference/applications/get-app-webapp-settings)**：返回 WebApp 的站点配置、主题和自定义选项。

## 收集反馈

- **[提交消息反馈](/zh/api-reference/feedback/submit-message-feedback)**：终端用户将回复评为 `like` 或 `dislike`，并可附加文字说明。
- **[获取应用的消息反馈](/zh/api-reference/feedback/list-app-feedbacks)**：返回应用内终端用户和管理员提交的全部反馈。

## 管理标注

标注为问题预设固定答案，命中时应用直接返回该答案，不再生成新回复：

- **[创建标注](/zh/api-reference/annotations/create-annotation)**、**[获取标注列表](/zh/api-reference/annotations/list-annotations)**、**[更新标注](/zh/api-reference/annotations/update-annotation)**、**[删除标注](/zh/api-reference/annotations/delete-annotation)**：管理标注集合。
- **[配置标注回复](/zh/api-reference/annotations/configure-annotation-reply)**：开启或关闭标注匹配。该变更异步执行。
- **[查询标注回复配置任务状态](/zh/api-reference/annotations/get-annotation-reply-job-status)**：使用返回的任务 ID 轮询，确认变更已完成。
