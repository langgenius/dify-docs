[文本生成应用](/zh/cloud/use-dify/build/text-generator) 运行在 `completion` 模式下，每次调用各自提交输入值并返回一条回复，调用之间不保留会话状态。

<Info>
认证方式、基础 URL，以及按终端用户区分数据的 `user` 字段，详见 [快速开始](/zh/api-reference/guides/get-started) 和 [终端用户身份](/zh/api-reference/guides/end-user-identity)。
</Info>

## 生成文本

- **[发送消息](/zh/api-reference/completion-messages/send-completion-message)**：将 `inputs` 中的取值填入应用的提示词模板并生成回复。阻塞式返回在生成完成后一次性返回全部内容；流式返回持续输出 `message` 事件，直至 `message_end`（应用开启文字转语音自动播放时还会收到 `tts_message` 事件）。
- **[停止响应](/zh/api-reference/completion-messages/stop-completion-message-generation)**：在回复完成前中断流式输出。

## 处理文件

- **[上传文件](/zh/api-reference/files/upload-file)**：上传图片、文档、音频或视频文件，供 [发送消息](/zh/api-reference/completion-messages/send-completion-message) 引用。文件仅限上传的终端用户使用。
- **[下载文件](/zh/api-reference/files/download-file)**：预览或下载已上传的文件，前提是该文件属于当前应用内的某条消息。
- **[获取终端用户信息](/zh/api-reference/end-users/get-end-user-info)**：根据终端用户 ID（例如上传文件响应中的 `created_by`）查询该用户的详细信息。

## 语音转写与合成

- **[语音转文字](/zh/api-reference/audio/convert-audio-to-text)**：将上传的音频文件（MP3、M4A、WAV、AMR 或 MPGA，最大 30 MB）转写为文字，让终端用户用语音代替打字输入。
- **[文字转语音](/zh/api-reference/audio/convert-text-to-audio)**：将生成的回复合成为语音。

## 获取应用信息与配置

- **[获取应用基本信息](/zh/api-reference/applications/get-app-info)**：返回应用的名称、描述、标签和模式。
- **[获取应用参数](/zh/api-reference/applications/get-app-parameters)**：返回调用需在 `inputs` 中传入的字段（名称、类型、默认值），以及应用的功能开关，可据此构建请求或客户端界面。
- **[获取应用元数据](/zh/api-reference/applications/get-app-meta)**：返回工具图标等配置元数据。
- **[获取应用 WebApp 设置](/zh/api-reference/applications/get-app-webapp-settings)**：返回 WebApp 的站点配置、主题和自定义选项。

## 收集反馈

- **[提交消息反馈](/zh/api-reference/feedback/submit-message-feedback)**：终端用户将回复评为 `like` 或 `dislike`，并可附加文字说明。
- **[获取应用的消息反馈](/zh/api-reference/feedback/list-app-feedbacks)**：返回应用内终端用户和管理员提交的全部反馈。
