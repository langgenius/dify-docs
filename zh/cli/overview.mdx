你在浏览器里构建 Dify 应用，但真正用到它们的场景往往在别处：一段 Shell 脚本、一条 CI 流水线，或是一个正在决定调用哪个工具的 AI Agent。

Dify CLI（difyctl）正是为此而生：只要能运行 Shell 命令，就能运行你的 Dify 应用。

## 你可以做什么

- 在终端中运行任意 Dify 应用，并获取其输出。
- 列出工作空间中的应用，查看每个应用所需的输入。
- 实时流式输出长响应，或获取结构化 JSON 用于脚本处理。
- 用 OAuth 登录一次，之后无需在脚本中存储密码或 API Key 即可自动化。
- 让 AI Agent 发现你的 Dify 应用并作为工具调用。

## 开始使用

首先 [安装 `difyctl`](/zh/cli/install)，然后按照 [快速开始](/zh/cli/quick-start) 登录并运行第一个应用。

<Tip>
想让 AI Agent 把你的 Dify 应用作为工具来运行？安装好 `difyctl` 后，从 [将你的 Agent 接入 Dify](/zh/cli/integrate-agents/overview) 开始。
</Tip>
