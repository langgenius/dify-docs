你的团队在浏览器里构建 Dify 应用，但真正用到它们的场景往往在别处：一段 Shell 脚本、一条 CI 流水线，或是一个正在决定调用哪个工具的 AI Agent。

Dify CLI（`difyctl`）填补了这一空白：只要能运行 Shell 命令，就能运行你的 Dify 应用。

## 你可以做什么

- 在终端中运行任意 Dify 应用，并获取其输出。
- 列出你可以运行的应用，查看每个应用所需的输入。
- 实时流式输出长响应，或获取结构化 JSON 用于脚本处理。
- 使用 Dify 账户或公司的单点登录（SSO）登录一次，之后无需在脚本中存储密码或 API Key 即可自动化。
- 让 AI Agent 发现你的 Dify 应用并作为工具调用。

## 开始使用

首先 [安装 `difyctl`](/zh/cli/install)，然后根据你的登录方式选择对应路径。
