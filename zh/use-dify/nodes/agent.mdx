自定义请求表单中显示的内容：

- 使用 **Markdown** 语法格式化和组织内容。

- 引用可用的**变量**来显示动态数据。

  例如，引用 LLM 节点的输出来展示 AI 生成的内容，供接收者审阅。

    <Tip>
        如果你引用推理模型的 `text` 输出变量，表单将显示模型的思考过程以及最终答案。
    
        如果只想显示答案，请为相应的 LLM 节点开启**启用推理标签分离**选项。
    </Tip>

- 添加交互式**输入字段**以从接收者处收集信息。

  输入字段可以使用变量预填充（显示建议内容）或静态文本预填充（提供示例或默认值）。接收者可以编辑预填充的内容或保持不变。

  每个输入字段都会成为一个变量，可以在下游节点中引用以整合接收者的输入。例如，将编辑后的内容传递给发布节点，或将反馈发送给 LLM 节点进行重新生成。
