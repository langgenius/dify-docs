自定义请求表单中显示的内容。

- **使用 Markdown 格式化和组织内容**

  使用标题、列表、粗体文本、链接和其他 Markdown 元素来清晰地呈现信息。

- **使用变量显示动态数据**

  引用 LLM 节点的输出供接收者审阅，或提供任何工作流数据作为上下文以便做出更明智的决策。

    <Tip>
        如果你引用推理模型的 `text` 输出变量，表单将显示模型的思考过程以及最终答案。
    
        如果只想显示答案，请为相应的 LLM 节点开启**启用推理标签分离**选项。
    </Tip>

- **使用输入字段收集信息**

  输入字段可以为空，或使用变量预填充（例如，供优化的 AI 生成文本）或静态文本预填充（例如，示例响应或默认值），接收者可以编辑这些内容。

  每个输入字段都会成为一个变量，可供下游节点引用。例如，将编辑后的内容传递给发布节点，或将反馈发送给 LLM 节点进行重新生成。
