将 [Dify 工具](/zh/use-dify/workspace/tools) 作为独立节点添加到你的工作流中。

这使你的工作流能够与外部服务和 API 交互，以访问实时数据并执行操作，例如网页搜索、数据库查询或内容处理。

**添加和配置工具节点**：

1. 在画布上，点击 **添加节点** > **工具**，然后从可用工具中选择一个操作。

2. 可选：如果工具需要认证，选择已有凭证或创建新凭证。

    <Info>
      要更改默认凭证，前往 **工具** 或 **插件**。
    </Info>

3. 完成其他所需的工具设置。
