<Note> ⚠️ 本文档由 AI 自动翻译。如有任何不准确之处，请参考[英文原版](/en/use-dify/knowledge/create-knowledge/import-text-data/readme)。</Note>

点击 Dify 平台顶部导航中的 **知识库** > **创建知识库** > **导入已有文本**。

拖拽或选中文件进行上传，**支持批量上传**，同时上传的文件数量限制取决于[订阅计划](https://dify.ai/pricing)。

**本地上传文档文件存在以下限制：**

* 单文档的上传大小限制为 **15MB**
* 不同 SaaS 版本的[订阅计划](https://dify.ai/pricing)限定了**批量上传个数、文档上传总数、向量存储空间**
