测试单个节点或逐步运行你的工作流以在发布前发现问题。

## 单节点测试

你可以单独测试任何节点而无需运行整个工作流。选择节点，在其设置面板中提供测试输入，然后点击运行以查看输出。

![](https://assets-docs.dify.ai/2025/04/376c9de6f92cb7a5f97a6661c5e0e9eb.png)

测试后，点击"上次运行"查看执行详情，包括输入、输出、时间和任何错误消息。

<Warning>
答案节点和结束节点不支持单节点测试。
</Warning>

## 逐步执行

当你一次运行一个节点时，它们的输出会缓存在变量检查器中。你可以编辑这些缓存的变量来测试不同场景，而无需重新运行上游节点。

![](https://assets-docs.dify.ai/2025/06/f8656d8deeeaefeab0a8d9169f0ed2d3.png)

当你想要测试节点如何响应不同数据而无需修改和重新运行它之前的所有节点时，这非常有用。只需在检查器中更改变量值并再次运行节点即可。

## 查看执行历史

每次节点执行都会创建一条记录。点击任何节点上的"上次运行"可查看其最近的执行详情，包括输入数据、输出内容以及执行时间。

![](https://assets-docs.dify.ai/2025/04/5ee92e6406979f5101d21865f95a86e5.png)
