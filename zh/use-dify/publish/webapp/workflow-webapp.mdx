工作流 Web App 将你的 Dify 工作流转换为生产就绪的应用程序，可以处理从单次运行到大规模批处理操作的各种任务。用户可以获得清晰的输入界面、实时处理反馈和全面的结果管理功能。

## 工作流应用如何工作

当你发布工作流时，Dify 会自动创建一个网页界面，该界面：

- **收集输入参数**：通过基于工作流起始变量的表单
- **处理请求**：使用你完整的工作流逻辑
- **处理结果**：具有内置的保存、复制和管理功能
- **自动扩展**：从单次运行扩展到批处理数百个项目

<Info>
与维护对话上下文的聊天应用不同，工作流应用设计用于产生特定输出的离散任务。
</Info>

## 单次执行

工作流应用的默认模式处理带有实时处理的单个请求：

<Frame>
  <img src="https://assets-docs.dify.ai/dify-enterprise-mintlify/en/guides/application-publishing/launch-your-webapp-quickly/4c5380cf71066d933082f7c30deacb01.png" alt="Single workflow execution interface" />
</Frame>

**用户体验：**
1. **填写输入表单** - 用户根据你工作流的起始变量提供参数
2. **点击运行** - 工作流执行，并显示实时进度指示
3. **查看结果** - 输出显示，可立即访问复制、保存和反馈选项
4. **采取操作** - 用户可以保存重要结果、提供反馈或生成类似输出

每个结果都包含内置操作：
- **复制** - 一键复制到剪贴板，方便分享
- **保存** - 将结果存储在应用的已保存项目中，供以后访问
- **反馈** - 点赞/点踩评级，帮助改进你的工作流
- **生成类似内容** - 基于当前结果生成变体（如果启用）

### 批处理优势

- **并行执行** - 多个工作流运行同时进行
- **进度跟踪** - 完成状态的实时更新
- **批量导出** - 完成后将所有结果下载为 CSV 文件
- **错误处理** - 失败的项目会清楚地标记错误详情

<Warning>
CSV 文件必须使用 Unicode 编码以防止导入失败。从 Excel 或类似工具保存时，请明确选择"Unicode (UTF-8)"编码。
</Warning>

## 结果管理

工作流应用包含全面的结果管理功能，帮助用户整理和重复使用输出：

### 保存结果

<Frame>
  <img src="https://assets-docs.dify.ai/dify-enterprise-mintlify/en/guides/application-publishing/launch-your-webapp-quickly/3cdd15e87aa1f1aae9f6abadb0f16d1f.png" alt="Saved results interface" />
</Frame>

**保存工作原理：**
- 用户点击任何想要保留的结果上的"保存"
- 保存的项目出现在专用的"已保存"标签中
- 每个保存的结果包含原始输入和完整输出
- 用户可以整理保存的结果并随时访问

<Info>
保存的结果在用户会话间持久保存，使工作流应用对构建个人输出库非常有用。
</Info>

### 生成变体

当你在工作流设置中启用"生成类似内容"时，用户可以生成成功结果的变体：

<Frame>
  <img src="https://assets-docs.dify.ai/dify-enterprise-mintlify/en/guides/application-publishing/launch-your-webapp-quickly/65fb111d8e89a8f7b761859265e42f0a.png" alt="More like this feature" />
</Frame>

**工作原理：**
1. 用户得到他们喜欢的结果
2. 他们点击"生成类似内容"来生成类似的输出
3. 工作流再次运行，带有轻微变化以产生不同但相关的结果
4. 用户可以迭代直到找到完美的输出

<Tip>
"生成类似内容"对于内容生成等创意工作流特别有效，用户希望探索同一主题的不同方法。
</Tip>

## 最佳实践

**对于单次执行应用：**
- 设计清晰的输入表单，指导用户提供有用信息
- 为复杂参数包含有用的描述
- 测试用户流程以确保结果符合预期

**对于批处理应用：**
- 提供清晰的 CSV 模板示例和样本数据
- 包含常见输入错误的验证
- 考虑非常大批次的处理限制

**对于结果管理：**
- 为产生用户想要保留的有价值输出的工作流启用保存
- 对创意或探索性工作流使用"生成类似内容"
- 考虑为需要外部访问的用户提供结果导出选项
