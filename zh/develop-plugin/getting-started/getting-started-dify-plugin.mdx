Dify 插件是模块化组件，通过外部服务、自定义功能和专业工具扩展你用 Dify 构建的 AI 应用。

<Frame caption="市场">
  ![市场](/images/develop-plugin/get-started/marketplace.png)
</Frame>

通过插件，你的 AI 应用可以：

- 连接外部 API
- 处理不同类型的数据
- 执行专业计算
- 执行现实世界的操作
