# アプリケーション公開

デバッグが完了したら、右上の「公開する」をクリックして、このワークフローを保存し、さまざまなタイプのアプリとして素早く公開することができます。

<figure><img src="https://assets-docs.dify.ai/img/jp/workflow/f8fb533bfc488d3eab087b9aa47280da.webp" alt=""><figcaption></figcaption></figure>

対話型アプリは以下の形式で公開できます：

* 直接実行
* Webサイトに埋め込む
* APIアクセス

ワークフローアプリは以下の形式で公開できます：

* 直接実行
* バッチ処理
* APIアクセス
* ツールとして公開

{% hint style="info" %}
複数バージョンのチャットフローやワークフローを管理する場合は、[バージョン管理](https://docs.dify.ai/ja-jp/guides/management/version-control)を参照してください。
{% endhint %}
