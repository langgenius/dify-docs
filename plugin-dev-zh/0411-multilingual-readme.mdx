Dify 插件支持多语言 README，并将在 [Dify Marketplace](https://marketplace.dify.ai) 等位置根据用户的首选语言展示对应的 README。

### README 文件规范

| 语种       | 必需 | 文件名                | 存放路径                  | 说明                |
| :------- | :-- | :----------------- | :-------------------- | :---------------- |
| **英语**   | 是  | `README.md`        | 插件根目录下                | /                 |
| **其他语种** | 否  | `README_<语言标识>.md` | 插件根目录下的 `readme` 文件夹中 | 目前支持日语、葡萄牙语、简体中文。 |

目录结构示例如下：

```bash
...
├── main.py
├── manifest.yaml
├── readme
│   ├── README_ja_JP.md
│   ├── README_pt_BR.md
│   └── README_zh_Hans.md
├── README.md
...
```

### Marketplace 展示规则

当插件提供了用户所用语言的 README 文件时，Dify Marketplace 的插件详情页将展示对应语言版本的 README。
