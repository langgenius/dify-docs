Difyは、AIワークフローを構築するためのオープンソースプラットフォームです。ビジュアルキャンバス上でAIモデルを編成し、データソースを接続し、処理フローを定義することで、ドメイン知識を直接動作するソフトウェアに変換できます。

<Info>
  Difyという名前は、**D**o **I**t **F**or **Y**ou（あなたのためにそれを行う）に由来しています。
</Info>
