# 使用云服务

{% hint style="info" %}
**提示:** Dify 目前正在 Beta 测试阶段，如文档与产品存在不一致，请以产品实际体验为准。

Dify 为所有人提供了[云服务](http://cloud.dify.ai)，你无需自己部署即可使用 Dify 的完整功能。要使用 Dify 云服务，你需要有一个 GitHub 或 Google 账号。

1. 登录 [Dify 云服务](https://cloud.dify.ai)，创建一个或加入已有的 Workspace
2. 配置你的模型供应商，或使用我们提供的托管模型供应商
3. 可以[创建应用](../guides/application-design/creating-an-application.md)了！


### 常见问题

**Q： 使用Dify Cloud时，我的数据是如何处理和存储的？**

A： 当您使用Dify Cloud时，您的用户数据会安全存储在位于美国东部地区的亚马逊服务器上。这包括您主动输入的数据和应用程序生成的任何数据。我们优先考虑您的数据的安全性和完整性，确保使用最高标准的云存储解决方案进行管理。

**Q： 采取了哪些措施来保护我的API密钥和其他敏感信息？**

A： 在Dify，我们了解保护您的API密钥和其他秘密的重要性。这些都是加密的，这意味着Dify无法查看它们，只有你，合法的所有者，才能访问你的秘密。

**Q： 你能解释一下应用程序数据是如何在Dify Cloud中匿名的吗？**

A： 在Dify Cloud中，我们对应用程序数据进行匿名处理，以确保隐私并减少加密和解密开销。这意味着应用程序使用的数据与可识别的用户帐户没有直接关联。通过匿名化数据，我们在保持云服务性能的同时增强了隐私。

**Q： 从Dify Cloud删除我的帐户和所有相关数据的过程是什么？**

A： 如果您决定删除您的帐户并从Dify Cloud中删除所有相关数据，您只需向我们的支持团队发送请求，邮箱为support@dify.ai.我们致力于尊重您的隐私和数据权利，根据要求，我们将根据数据保护法规从我们的系统中删除您的所有数据。
